# Product

## Register

product

## Users

Researchers, analysts, and technically-minded builders who run AI-driven research on their own machine and do not trust raw model output. They supervise long runs (minutes to hours) at a desk, read manuscripts and gate verdicts carefully, and intervene occasionally. Some are beginners guided by the Tutorial tab.

## Product Purpose

Nullius is an evidence-gated research OS: AI agents plan, write and execute analysis code in a sandbox, while deterministic gates (numeric grounding, citation verification, reproducibility, write-boundary blocking) prevent any unverified claim from entering the manuscript. Success = a user ships a report whose every number and citation is traceable, without reading a single fabricated fact.

## Brand Personality

Skeptical, precise, calm. The interface is a quiet scientific instrument: it observes, verifies, and stays out of the way. Authority comes from legibility and restraint, not from dashboards shouting.

## Anti-references

- Hacker-terminal dark consoles (tried; clashed with the app's character — rejected by the owner).
- SaaS dashboard clichés: gradient hero metrics, cream backgrounds, identical card grids.
- Crypto-trading-desk density and blinking noise.

## Design Principles

1. **The manuscript is the hero.** The produced paper reads like a paper; the UI around it recedes.
2. **Color is state, never decoration.** Green = verified, amber = needs attention, red = blocked, indigo = actionable. Nothing else is colored.
3. **Trust through legibility.** High contrast, generous reading measure, mono for data, serif for the artifact.
4. **One vocabulary everywhere.** Same buttons, badges, and status dots on every screen.
5. **Supervision, not spectacle.** Live activity informs the user's next decision; no orchestration, no theater.

## Accessibility & Inclusion

Body text ≥4.5:1 contrast; focus-visible rings on all interactive elements; `prefers-reduced-motion` honored (caret blink and transitions collapse to static); English UI with bilingual (EN/JA) tutorial.
